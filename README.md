# FocusAny 插件开发示例

## 基础示例

### BasicExample

原生 HTML、CSS、JS 开发，无任何框架，可以查看到最基础的插件开发方式。

### FileEditorExample

基于 vue3 + vite + tailwindcss 开发的基础文件编辑器，可以使用很轻量的方式集成一些文件编辑器。

### ReactExample

基于 react + vite + tailwindcss 开发的插件示例。

### VueExample

基于 vue3 + vite + tailwindcss 开发的插件示例，可以查看到如何使用 vue3 开发插件。

### WebRemoteExample

将远程网页快速变为本地离线缓存的插件示例。
